# ✦ PageCraft Pro
### AI 상품 상세페이지 자동 생성기 — 패션 · 의류 · 잡화 특화

> 상품 사진을 올리면 AI가 카피라이팅부터 상세페이지 이미지까지 자동으로 만들어줍니다.

## 🚀 배포 방법 (Vercel)

> 총 소요 시간: 약 10분  
> 비용: **무료** (Vercel 무료 플랜으로 충분합니다)

---

### STEP 1 — GitHub 계정 만들기

아직 GitHub 계정이 없다면 먼저 만들어주세요.

1. [github.com](https://github.com) 접속
2. **Sign up** 클릭
3. 이메일 인증까지 완료

---

### STEP 2 — 새 저장소(Repository) 만들기

1. GitHub 로그인 후 오른쪽 상단 **`+`** 버튼 클릭
2. **New repository** 선택
3. 아래와 같이 설정:

```
Repository name: pagecraft-pro
설명 (선택): AI 상품 상세페이지 생성기
공개 설정: Public ✅  (Vercel 무료 배포를 위해 Public 권장)
```

4. **Create repository** 클릭

---

### STEP 3 — 파일 업로드

1. 압축 파일(`PageCraft_Pro_배포파일.zip`)을 PC에서 압축 해제
2. 아래 파일 5개를 확인합니다:

```
📁 pagecraft-pro/
├── 📄 index.html          ← 메인 웹앱
├── 📄 vercel.json         ← 배포 설정
├── 📄 package.json        ← 패키지 정보
├── 📄 README.md           ← 이 파일
└── 📁 api/
    ├── 📄 generate.js     ← AI 카피 생성
    └── 📄 render.js       ← 이미지 렌더링
```

3. GitHub 저장소 페이지에서 **Add file → Upload files** 클릭
4. `api` 폴더를 포함한 **모든 파일** 드래그하여 업로드
5. **Commit changes** 클릭

> ⚠️ **주의**: `api` 폴더 안의 파일도 반드시 함께 올려야 합니다!

---

### STEP 4 — Vercel에 배포하기

1. [vercel.com](https://vercel.com) 접속
2. **GitHub으로 로그인** 클릭
3. **New Project** 버튼 클릭
4. 저장소 목록에서 `pagecraft-pro` 선택 → **Import** 클릭
5. 설정 변경 없이 바로 **Deploy** 클릭
6. 배포 완료까지 약 1~2분 대기

---

### STEP 5 — API 키 설정 ⭐ 가장 중요!

> 이 단계를 빠뜨리면 AI 기능이 작동하지 않습니다.

1. Vercel 대시보드 → 방금 만든 `pagecraft-pro` 프로젝트 클릭
2. 상단 탭에서 **Settings** 클릭
3. 왼쪽 메뉴에서 **Environment Variables** 클릭
4. 아래 정보 입력 후 **Save** 클릭:

```
Name  : ANTHROPIC_API_KEY
Value : sk-ant-여기에_발급받은_키_입력
```

5. 다시 **Deployments** 탭으로 이동
6. 최신 배포 항목 오른쪽 **`⋯`** 클릭 → **Redeploy** 클릭
7. 재배포 완료 후 정상 작동!

---

### STEP 6 — 링크 확인 및 공유

배포가 완료되면 아래와 같은 링크가 생성됩니다:

```
https://pagecraft-pro-여러분이름.vercel.app
```

이 링크를 수강생분들과 공유하시면 됩니다! 🎉  
별도 설치 없이 브라우저에서 바로 사용 가능합니다.

## 🔑 API 키 발급

### Anthropic 콘솔에서 발급받는 방법

1. [console.anthropic.com](https://console.anthropic.com) 접속
2. 회원가입 또는 로그인
3. 왼쪽 메뉴 **API Keys** 클릭
4. **Create Key** 버튼 클릭
5. 키 이름 입력 (예: `pagecraft-pro`)
6. 생성된 `sk-ant-...` 키를 복사

> ⚠️ **주의사항**
> - API 키는 한 번만 표시됩니다. 반드시 복사해서 안전한 곳에 보관하세요.
> - API 사용량에 따라 요금이 발생합니다.
> - 수강생이 많을 경우 월 사용 한도를 설정하는 것을 권장합니다.

### 요금 참고 (2025년 기준)

| 모델 | 입력 | 출력 |
|------|------|------|
| Claude Sonnet 4 | $3 / 1M 토큰 | $15 / 1M 토큰 |

> 상세페이지 1개 생성 시 약 $0.01~0.03 수준입니다.

## 📖 사용 방법

### 기본 사용 순서

```
1. 상품 사진 업로드 (드래그 또는 클릭)
       ↓
2. 브랜드명 · 상품명 · 가격 입력
       ↓
3. 카테고리 · 플랫폼 선택
       ↓
4. 강조하고 싶은 특징 태그 선택 (방수, 경량 등)
       ↓
5. [AI 상세페이지 생성] 버튼 클릭
       ↓
6. 미리보기 확인 후 PNG 이미지 저장
   또는 텍스트 복사하여 쿠팡/스마트스토어에 등록
```

### 각 입력 항목 설명

| 항목 | 필수 | 설명 |
|------|------|------|
| 상품 사진 | 권장 | AI가 이미지를 분석해 더 정확한 카피를 생성합니다 |
| 브랜드명 | 선택 | 없으면 AI가 이미지에서 판단합니다 |
| 상품명 | 선택 | 없으면 AI가 자동으로 생성합니다 |
| 판매 가격 | 선택 | 입력 시 상세페이지 하단에 가격이 표시됩니다 |
| 카테고리 | 필수 | 패딩, 슬리퍼, 스카프 등 선택 |
| 플랫폼 | 필수 | 쿠팡 또는 스마트스토어 선택 |
| 강조 특징 | 선택 | 방수, 경량, 남녀공용 등 해당되는 태그 클릭 |
| 추가 설명 | 선택 | 색상, 소재, 특이사항 등 자유롭게 입력 |

### 결과물 활용 방법

**PNG 이미지 저장**
- 우측 패널 → 내보내기 탭 → "이미지 저장 (PNG)"
- 쿠팡 윙 / 스마트스토어 상세페이지 이미지 섹션에 직접 업로드

**텍스트 복사**
- 우측 패널 → 복사용 텍스트 탭
- 상품명, 카피, 스펙 등 항목별로 복사 가능
- 쿠팡 상품 등록 시 각 필드에 붙여넣기

## ❓ 자주 묻는 질문

**Q. 배포 후 "API 키가 설정되지 않았습니다" 오류가 나요**  
A. STEP 5의 환경변수 설정을 완료했는지 확인하고, Redeploy를 한 번 더 눌러주세요.

**Q. 사진 없이도 사용할 수 있나요?**  
A. 네! 상품명과 카테고리만 입력해도 AI가 카피를 생성합니다. 단, 사진이 있을 때 더 정확합니다.

**Q. 생성된 이미지가 깨져 보여요**  
A. `api/render.js`의 서버 렌더링 실패 시 텍스트 미리보기로 대체됩니다.  
   이 경우 우측 복사용 텍스트를 활용하시면 됩니다.

**Q. 수강생이 많은데 비용이 걱정돼요**  
A. console.anthropic.com → Settings → Limits에서 월 최대 사용 금액을 제한할 수 있습니다.

**Q. 쿠팡/스마트스토어 이외 플랫폼도 지원하나요?**  
A. 네! 플랫폼 선택란에서 선택한 플랫폼에 맞게 카피 톤이 자동으로 조정됩니다.

**Q. 생성된 내용을 수정할 수 있나요?**  
A. 우측 패널의 텍스트를 복사 후 직접 수정하여 사용하시면 됩니다.
